# 宠物领养系统 java674

## 项目概述

宠物领养系统是一款集流浪动物管理、救助管理与宠物领养于一体的管理系统。该系统实现了前后端分离，基于 Java 技术栈开发，具有高度模块化和可扩展性。

## 技术栈

- 前端：Vue
- 后端：Java、SpringBoot
- 数据库：Mysql

## 运行环境

- 系统环境：Windows、Mac
- 开发环境：IDEA、JDK1.8、Mysql

## 功能模块

### 用户端功能

- **首页**：展示公告列表、宠物科普、流浪宠物介绍以及热门活动。
- **宠物领养**：用户可搜索、申请领养宠物并查看已领养宠物信息。
- **宠物救助**：查看需救助的流浪宠物信息，添加新流浪宠物信息，展示宠物救助站信息。
- **丢失宠物查看**：查看丢失宠物及失主信息。
- **宠物论坛**：交流领养心得，评论他人帖子。

### 管理员端功能

- **主页**：展示各季度注册人数，以图表形式展示数据。
- **用户管理**：管理注册用户信息。
- **角色管理**：为用户分配系统菜单。
- **菜单管理**：后台菜单管理。
- **文件管理**：系统文件统一管理。
- **流浪动物管理**：发布流浪宠物信息。
- **待绝育动物管理**：管理流浪动物健康信息。
- **申请领养管理**：处理用户领养信息。
- **评论管理**：管理论坛评论。
- **流浪动物救助**：发布需救助的流浪动物信息，处理已解决信息。
- **喂养点管理**：管理宠物喂养点。
- **动物走失管理**：管理丢失动物信息。
- **救助站管理**：管理流浪动物救助站。
- **帖子管理**：管理宠物论坛帖子。
- **捐赠管理**：添加捐赠人信息。
- **公告管理**：管理首页公告。
- **科普文章管理**：管理科普文章展示。
- **活动管理**：管理宠物相关活动。

## 部署步骤

1. 准备运行环境：Windows 或 Mac，安装 IDEA、JDK1.8、Mysql。
2. 导入数据库文件：执行 `database/schema.sql` 和 `database/data.sql`。
3. 编译后端代码：进入 `backend` 目录，执行 `mvn clean package`。
4. 部署前端代码：进入 `frontend` 目录，执行相关构建命令。
5. 启动后端服务：执行后端可执行文件或服务。
6. 访问系统：通过浏览器访问前端页面。
